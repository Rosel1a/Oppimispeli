# Oppimispeli

Oppimispeli on verkkopohjainen oppimispeli / harjoitusjärjestelmä, jonka tarkoituksena on tarjota oppijoille interaktiivinen alusta harjoitteluun ja opiskeluun. Projekti on toteutettu Pythonilla ja sisältää HTML/CSS-käyttöliittymän.

## Ominaisuudet

- Web-sovellus, jossa on käyttöliittymä HTML ja CSS -tiedostoilla  
- Backend Pythonilla 
- Tietokantatoiminnallisuus harjoitusten ja käyttäjien tallennukseen  
- Staattiset resurssit ja templates-hakemistot käyttöliittymän monipuolistamiseksi  
- Mahdollisuus laajentaa harjoituksia, kysymyksiä ja sisältöä helposti
